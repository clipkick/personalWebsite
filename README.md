> This is my personal website. It runs on nodejs and react using mongodb for a backend database.

## Install

```sh
npm install
```

## Usage

> run dev server, webpack, scss and the node server

```sh
npm run dev
npm run scss
npm start
```

> compile files for production

```sh
npm run prod
npm run buildClient
npm run buildServer
```

> run prod server

```sh
npm run startProd
```
